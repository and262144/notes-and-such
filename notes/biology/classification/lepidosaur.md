## aka reptiles!

| Taxon       | Name                                 |
|-------------|--------------------------------------|
| Kingdom:    | **Animalia**                         |
| Phylum:     | **Chordata**                         |
| Class:      | **Sauropsida**                       |
| Subclass:   | **Diapsida**                         |
| Infraclass: | **Lepidosauromorpha**                |
| Superorder: | **Lepidosauria**                     |

- Lepidosauria are tetrapod vertebrate
- 12,000 known species
- Found in every continent except Antartica
